What is the DC208 network?

DC208 started as an offshoot of the defcon groups project. The goal was to create a banner under which local groups could collect and celebrate the hacker traditions.

In case it wasn't obvious, 208 is the area code for the state of Idaho (Until 2016, when we ended up with 2 area codes!) Idaho is quite large though, so one banner is insufficient to capture all that is going on here. This site is built to be a resource for Hackers, Makers, Crafters, InfoSec, and the curious within and around Idaho.
